# 采购调研填写系统使用说明

## 给采购同事的填写方式

请打开同目录下的文件：

`index.html`

打开后按左侧模块逐项填写。系统会自动保存在当前电脑的当前浏览器中，填写过程中不用手动保存。

## 填写要求

1. 请优先填写过去 1-2 年实际采购过、今年仍计划采购的水果。
2. 能填写具体水果、产地、时间、供应商、价格的，请尽量写具体。
3. 不确定的信息请填写“待确认”，不要猜。
4. 没有记录的信息请填写“目前无记录”。
5. 涉及敏感联系人或价格时，可以只填写内部可共享内容。

## 回收方式

填写完成后，请点击页面右上角：

- `导出数据`：生成 JSON 文件，建议作为正式回收文件。
- `导出表格`：生成 CSV 文件，便于人工查看和初步汇总。
- `打印`：可打印或另存为 PDF。

建议采购提交：

1. JSON 文件
2. CSV 文件
3. 如有历史 Excel、报价表、供应商名单、采购复盘等附件，请一起提交

## 注意事项

1. 自动保存只保存在当前电脑的当前浏览器中。
2. 换电脑或换浏览器后，需要用此前导出的 JSON 文件重新导入。
3. 点击 `清空` 会删除当前浏览器里已保存的填写内容，请谨慎操作。
4. 若多人填写，请每个人单独导出自己的 JSON 文件，不要共用同一个文件继续填写。
